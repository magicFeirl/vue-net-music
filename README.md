## VUE 仿写网易云音乐

### 运行

```bash
yarn install # 安装依赖
yarn serve
```

### 实现的页面

* 首页

* 推荐歌单

* 最新MV
* 专辑页

### 实现的功能

* 点击MV可以跳转到对应页面观看MV
* 点击歌曲名可以播放音乐
* 点击专辑名可以跳转到对应专辑页
* 可调整音乐播放时间、音量、是否暂停播放
* ...

### 用到的轮子

* vue-slider-component 滑块组件，用来实现 player bar
* element-ui
* axios
* vuex
* vue-router
* [网易云音乐 NodeJS 版 API](https://neteasecloudmusicapi.vercel.app) 
* ...
